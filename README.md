CookBook

Descripción

CookBook es una aplicación móvil desarrollada con React Native CLI para la asignatura de Desarrollo Móvil, la app permite gestionar recetas de cocina de manera práctica y visual, con una experiencia de usuario clara e intuitiva.

Funcionalidades

-Explorador de Recetas: muestra una lista de recetas disponibles, al seleccionar una receta, se despliega una pantalla de detalle con ingredientes y pasos de preparación.
-Agregar nueva receta: incluye un formulario sencillo para registrar nuevas recetas, el formulario valida que todos los campos requeridos estén completos.
-Tips de Cocina: presenta consejos de cocina que rotan automáticamente cada 5 segundos para facilitar el aprendizaje de trucos culinarios.

Tecnologías

-React Native CLI
-React Navigation (Tab Navigation y Stack Navigation)
-Hooks de React (`useState`, `useEffect`)
-Estilos con `StyleSheet`

Notas del proyecto

-La interfaz de usuario está en español.
-La lógica interna del proyecto se mantiene en inglés para facilitar la lectura del código.

Instalación y ejecución

1.Clonar el repositorio.
2.Abrir una terminal en la carpeta raíz del proyecto.
3.Ejecutar `npm install` para instalar las dependencias.
4.Iniciar el emulador o dispositivo de prueba.

Android


npx react-native run-android


iOS


npx react-native run-ios
